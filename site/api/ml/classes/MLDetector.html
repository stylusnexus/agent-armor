    Class MLDetector

    ONNX-based detector running the DeBERTa-v3-small multi-label classifier. Implements Agent Armor's Detector interface via structural typing — MLDetector.create loads the model; MLDetector.scanAsync runs inference. MLDetector.scan (sync) always returns no threats, since ONNX inference is async-only.
